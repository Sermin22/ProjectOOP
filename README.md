# Использование классов и объектов на основе популярной темы e-commerce

## Модуль product_and_category:

В модуле реализованы классы Product и Category.

Для класса Product определены следующие свойства:
- название (name),
- описание (description),
- цена (price),
- количество в наличии (quantity).

В классе Product создан класс-метод new_product, который принимает на вход параметры 
товара в словаре и возвращает созданный объект класса Product.

Атрибут цены в классе приватный и содержит геттеры и сеттеры. В сеттере реализована 
проверка: в случае если цена равна или ниже нуля, выводите сообщение в консоль 
“Цена не должна быть нулевая или отрицательная”, при этом новая цена не устанавливается.

Для класса Category определены следующие свойства:

- название (name),
- описание (description),
- список товаров категории (products).

В классе Category список товаров приватный атрибут, чтобы к нему нельзя было получить 
доступ извне. Для добавления товаров в категорию реализован специальный метод 
add_product(), в который передается объект класса Product и уже он записывается 
в приватный атрибут списка товаров.

В классе Category реализован геттер, который выводит список товаров в виде строк 
в формате:

Название продукта, 80 руб. Остаток: 15 шт.



## Описание:

Для этих двух классов добавлена инициализация, где каждый параметр передан 
при создании объекта и сохранен.

Также для класса Category добавлены два атрибута класса category_count и 
product_count. Доступ к ним есть у каждого объекта класса, и в них хранится 
общая информация для всех объектов. Эти атрибуты хранят в себе количество 
категорий и количество товаров. Атрибуты класса заполняются автоматически 
при инициализации нового объекта, считается не количество на складе, а длина 
списка с товарами.

## Модуль heirs_product

В модуле выделены две категории товаров и созданы под них классы:
### Класс «Смартфон» (Smartphone)
Помимо имеющихся свойств, в него добавлены следующие:
- производительность (efficiency),
- модель (model),
- объем встроенной памяти (memory),
- цвет (color).

### Класс «Трава газонная» (LawnGrass)
Помимо имеющихся свойств, в него добавлены следующие:

- страна-производитель (country),
- срок прорастания (germination_period),
- цвет (color).

Оба класса Smartphone и LawnGrass, это классы-наследники от исходного класса 
Product.
